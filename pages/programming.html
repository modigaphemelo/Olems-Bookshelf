Language Foundations

Modern, beginner-friendly intros to core programming languages.

Modern C for Absolute Beginners by Slobodan Dmitrović (2021)

Overview

C stripped to essentials—no legacy cruft. Dmitrović’s crash course is crisp, example-rich, and beginner-friendly, but pointer-phobes beware: it’s a gentle nudge, not a deep dive. 8/10.

C++ Without Fear by Brian Overland (2016)

Overview

C++ tamed for the timid—Overland’s hand-holding tour dodges the terror of templates and pointers. Solid for newbies, but pros yawn at the baby steps. 7/10.

Web & Markup

Building the front-end: HTML, CSS, and lightweight docs.

HTML & CSS: Design and Build Websites by Jon Duckett (2011)

Overview

Duckett’s visual masterpiece—HTML/CSS as art book. Stunning layouts, crystal-clear code, but it’s 2011: no Flexbox, no Grid. Still the gold standard for beauty in teaching. 9/10.

The Markdown Guide by Matt Cone (2019)

Overview

Markdown in 100 pages—Cone’s cheat sheet is clean, free, and exhaustive. Perfect for writers, useless for coders who already live in `.md`. 6/10.
